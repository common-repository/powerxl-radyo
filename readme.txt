== Description ==

Bu eklenti sayesinde Wordpress blogunuzda PowerXL Radyo kurulmaktadır.

== Installation ==

1. Wordpress Admin panelinde, EKLENTİLER > Yeni EKLE ve  'PowerXL' olarak arama yapın.

2. 'Yükle' Tıklayın.

3. Wordpress Admin içinde , Görünüm > Bileşenler, içinde  'PowerXL Radyo’ eklentisini istediğiniz bölüme sürükleyip yerleştirin ve kaydedin..
